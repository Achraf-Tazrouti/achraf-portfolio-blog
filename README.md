# Portfolio Blog

Dit project is een stageblog waar ik elke week een nieuwe blogpost plaats over wat ik tijdens mijn stage heb geleerd.  
De applicatie is gebouwd als oefening in Angular, een framework dat ik tijdens mijn stage heb geleerd gebruiken.  
De backend is gemaakt met Node.js en Express.

- Elke week voeg ik een nieuwe blogpost toe.
- De site toont mijn leerproces en technische groei tijdens de stage.
- Dit project is bedoeld als portfolio en stageverslag.
